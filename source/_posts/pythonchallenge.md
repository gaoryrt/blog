        print '\tModified:\t', datetime.datetime(*info.date_time)

# [第十关](//www.pythonchallenge.com/pc/return/bull.html)
1 11 21 1211 111221 ...  
求`len(a[30])`  
1:一个一  
11:两个一   
21:一个二一个一  
1211:一个一一个二两个一  
这个顺序来的  
直接写程序就行了:  
```
def next(arr):
    last = arr[0]
    count = 1
    s =''
    for i in arr[1:]:
        if last == i:
            count += 1
        else:
            s += str(count)+last
            last = i
            count = 1
    s += str(count)+last
    print len(s)
    return s

arr = "1"
for i in range(30):
    arr = next(arr)
```
我代码写的不好，但是正确了的。。。  
把上面的`print s`改成`print len(s)`就可以得出答案了
